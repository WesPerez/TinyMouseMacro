# 轻量鼠标宏

一个很轻量的 Windows 鼠标宏工具：

- 可以保存多个配置。
- 每个配置有一个全局快捷键。
- 动作只保留三种：移动、左键点击、等待。
- 配置文件保存在程序目录下的 `macros.json`。

## 界面说明

- `X 坐标`：鼠标要移动或点击的横向坐标。
- `Y 坐标`：鼠标要移动或点击的纵向坐标。
- `等待毫秒`：添加等待动作时使用，例如填 `500` 就等待半秒。
- `添加移动动作`：只把鼠标移动到 X/Y，不点击。
- `添加点击动作`：移动到 X/Y 后左键点击一次。
- `添加等待动作`：在步骤里插入一段等待时间。
- `刷新坐标`：刷新底部显示器坐标信息，方便确认每块屏幕的范围。

## 使用方式

1. 运行 `TinyMouseMacro.exe`。
2. 左侧新增或选择一个配置。
3. 填快捷键，例如 `Alt+Z`、`Ctrl+Alt+F8`、`Win+Shift+Q`。
4. 点 `录制下一次点击`，然后在目标位置左键点一下；也可以手动填 X/Y 后点 `添加移动动作` 或 `添加点击动作`。
5. 点 `保存并启用`，状态栏提示快捷键已注册后就能使用。

## 多显示器说明

工具使用 Windows 虚拟屏幕坐标，副屏在主屏左侧或上方时可能出现负数坐标。底部会显示虚拟屏幕范围和每个显示器的坐标范围。

快捷键使用 Windows 全局热键，并加了低级键盘监听作为后备。正常情况下，只要宿主 Windows 能收到组合键，所有显示器都能触发。如果 VMware、远程桌面或其他全屏程序开启了独占键盘捕获，它们仍然可能先吃掉快捷键。
